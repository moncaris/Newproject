Landing Page Project

This project is a basic HTML and CSS landing page. It includes a simple design with a title, a paragraph, and a button. The page is styled using internal CSS and is centered both vertically and horizontally within the viewport.

Technologies Used:
- HTML5
- CSS3

To view the page, simply open the index.html file in a web browser.

Alternatively, the project can be containerized and run using Docker, as outlined in the provided Dockerfile.
